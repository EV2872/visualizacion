# Prácticas de la asignatura visualización

Cada carpeta de practicas tiene su propio README.md file

## Dependencias comunes a cada proyecto
Crear el entorno virtual de python
```console
[python|python3] -m venv visualizacionVEM
```

Activar el entorno virtual
```console
source visualizacionVEM/bin/activate
```

Instalar dependencias
```console
pip install -r requirements.txt
```

Desactivar el entorno virtual
```console
deactivate
```
